# Roadmap

Glyph Hold is usable as a self-hosted memory and secrets service for AI agents.
This roadmap tracks the public areas most likely to improve next.

## Near Term

- Improve agent guidance for writing useful memories: clear titles, concise
  summaries, stable categories, confidence scores, and good tags.
- Strengthen duplicate and conflict handling before agents create new memories.
- Expand examples for the Glyph Hold MCP server and Hermes memory provider.
- Make dashboard help text clearer around summaries, body text, confidence, and
  prefetch levels.

## Later

- Add more operational examples for common self-hosted deployments.
- Continue improving recovery, backup, and upgrade documentation.
- Refine dashboard workflows based on live agent usage.
- Add integration notes as new agent connectors mature.

## Notes

GitHub Releases are the source of truth for version-specific release notes. This
file is intentionally about upcoming direction, not a changelog.
